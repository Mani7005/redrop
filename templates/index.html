SQL UPDATE + BEFORE Trigger

Update Donor's Donation Date

Fires trigger trg_check_availability (BEFORE UPDATE) which automatically recalculates donor availability based on the 90-day cooldown rule

⚡

Trigger: trg_check_availability (BEFORE UPDATE on donors)

Fires on every UPDATE. Uses DATEDIFF(CURDATE(), NEW.last_donation): if < 90 days → is_available = FALSE, else → TRUE. Transparent to the application layer.

Trigger Source Code

BEFORE UPDATE ON donors
FOR EACH ROW
BEGIN
  IF NEW.last_donation IS NOT NULL THEN
    IF DATEDIFF(CURDATE(),
         NEW.last_donation) < 90
    THEN
      SET NEW.is_available = FALSE;
    ELSE
      SET NEW.is_available = TRUE;
    END IF;
  END IF;
END
